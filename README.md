# **BNP V1.3**
#### 一款可输入富文本的记事本app 

##### 部分功能还在更新中

### **关键库引用**

[TakePhoto](https://github.com/crazyandcoder/TakePhoto)<br>
[Icarus](https://github.com/mr5/icarus-android)


### **功能点**

 1. 支持本地选择照片以及相机拍摄输入图片文本
 2. 支持富文本编辑
 3. 支持代码块插入

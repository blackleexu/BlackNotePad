# **BNP V1.3**
#### 一款可输入富文本的记事本app 
<br> 
### 引用关键库：
[TakePhoto](https://github.com/crazyandcoder/TakePhoto)<br> 
[Icarus](https://github.com/mr5/icarus-android)<br> 


### **功能点**

 1. 支持本地选择照片以及相机拍摄输入图片文本
 2. 支持富文本编辑
 3. 支持代码块插入
